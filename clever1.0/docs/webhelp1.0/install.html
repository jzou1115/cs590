Installation

Note: these installation instructions are written for Unix users, but Windows versions of the GenSeqs, CETrFile, and CEEnergy binary files are provided in clever1.0/compiled/clever-1.0-windows.zip.

1. Run

    unzip clever_package.zip
    The resulting directory structure is
       clever1.0/
                      compiled/
                      docs/
                      docs/webhelp1.0/
                      source/
                      test/
                      tutorial/

2. Obtain executable files

     a. Using compiled files
            Change directory to clever1.0/compiled and run tar xzvf clever-1.0-linux-binary.tar.gz
            This unzips the binary files GenSeqs, CETrFile, and CEEnergy.
     b. Using source codes
            Change directory to clever1.0/source and type “make” (code was compiled with g++3.4.6 version)
            This generates GenSeqs, CETrFile, and CEEnergy.

3. Copy the binary files (GenSeqs, CETrFile, and CEEnergy) to an appropriate directory.
